# MyEncyclopedia

This repo does not contain any code. 
The entire body of work is focused in the wiki section of this repo. 

#### Purpose
The wiki serves as a collection of knowledge I have learned and gathered throughout the years. While studying at Concordia University, I quickly realised how massive the Software domain is, and how diverse the knowledge pool is. There is so much to learn that often times, when starting a new subject, the little details of the previous one slowly start slipping away from memory. This body of work is an attempt to avoid such predicament, and is intended to be used as a coding helper, much like how an actual encyclopedia serves as a keeper of knowledge for future recalls.

#### Dynamic Nature of the Repo
It is important to note that this body of work is not static. It is rather a compounding collection of different subjects slowly growing over the years. If there are obvious (to you) missing pieces of information in the wiki, it is only because I have not yet written it for whatever reason. The time will come, and it will be eventually added.

#### Future Works and Ideas.
